About app

Application is designed for gathering invoices as pdf attachments.

This files sent by emails are downloaded and stored in device memmory.

Application is extracting from pdfs information about payment deadline, amount to pay and account number for transfering money.

User creates an account, and provides his email addresses and mailbox passwords.

This data allows the app to login to user mailbox and check for new emails with invoices.

User creates trusted list, which is a list of trusted invoice senders.

All data are stored in Firebase database. Sensitive data is encrypted.

All of its items can be named.

Most important Features

-Creating and login to app account.

-Downloading attachments from multiple user mailboxes.

-Displaying invoices as Tiles.

-Payments dates as events marked on calendar.

-Undefined tab contains invoices, which informations can't be extracted correctly.

-Possibility to view downloaded PDF file with option of editing parsed data.

-Modern UI.
